SUPPORTED LINUX:-

Debian-based systems (e.g., Ubuntu, Debian):

Uses apt-get as the package manager.
Typically uses systemd as the init system, but older versions may use init.d.
Red Hat-based systems (e.g., CentOS, Red Hat Enterprise Linux, Fedora):

Uses yum or dnf as the package manager.
Uses systemd as the init system.
SUSE-based systems (e.g., openSUSE, SUSE Linux Enterprise):

Uses zypper as the package manager.
Uses systemd as the init system.
Arch-based systems (e.g., Arch Linux, Manjaro):

Uses pacman as the package manager.
Uses systemd as the init system.
